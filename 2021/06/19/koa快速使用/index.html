koa快速使用

邢火林 2021-06-19 11:54:29
Categories： Tags：

KOA2

单独使用koa插件

初始化项目

npm init

安装koa2

npm install koa --save

koa2环境搭建

全局安装脚手架

npm install -g koa-generator

查看版本

koa2 --version

创建项目

koa2 '项目名称'

启动项目

npm run dev

访问

localhost：3000

koa 路由创建方式

创建路由

const router = require('koa-router')();
//设置前缀
router.prefix('/api')
router.get('/list',function(ctx,next){
  ctx.body = 'api list'
})

router.post('create',function(ctx,next){
  ctx.body ='api create'
})

module.exports = router

app.js 中引入路由
const comments = require('./router/comments')

服务器自动重新部署：nodemon

npm i nodemon -g

启动 nodemon 代替 node
